# 分析输出文件目录

* **文件夹U13PowerUpData中包含了多次使用逻辑分析仪对U13IC抓取的上电数据。**
* **文件夹SerialOutput中包含串口输出的启动信息。**
* **文件NetAnalyzeV0.4.md中包含主要IC的电路连接逻辑。**



## U13研究：

  **U13电路连接情况：**

| 1 : 3V3                             | 8 : GND               |
| ----------------------------------- | --------------------- |
| 2 : To --> U1:49 / PE0              | 7 : To -- > CON1 - 37 |
| 3 : To --> CON1 - 40                | 6 : To --> CON1 - 39  |
| 4 : To --> U1:53 / PE5 (10K to 3V3) | 5 : To --> CON1 - 38  |

 

**逻辑分析仪各通道连接情况与功能猜测：**

| 逻辑分析仪通道 | U13脚位 | 功能猜测  |
| -------------- | ------- | --------- |
| 0              | Pin 4   | SCL       |
| 1              | Pin 2   | SDA       |
| -              | -       | -         |
| 2              | Pin 7   | CS/UnKnow |
| 3              | Pin 5   | MOSI      |
| 4              | Pin 6   | CLK       |
| 5              | Pin 3   | MISO /CS  |
